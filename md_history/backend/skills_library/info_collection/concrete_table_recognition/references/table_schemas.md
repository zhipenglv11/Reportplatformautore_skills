# 表格Schema定义

本文档定义了各表格类型的数据结构和字段说明。

## 混凝土回弹检测记录表（结构物检测原始记录）

### JSON Schema

```json
{
  "type": "object",
  "properties": {
    "检测日期": {
      "type": "string",
      "description": "表格右上角「检测日期」字段，格式如：2024-10-10"
    },
    "检测原因": {
      "type": "string",
      "description": "表格中部靠右的「检测原因」字段，如：委托检测"
    },
    "检测方法": {
      "type": "string",
      "description": "表格中部靠右的「检测方法」字段，如：回弹法"
    },
    "检测部位": {
      "type": "string",
      "description": "从「工程名称」或「检测部位」字段提取，如：2#楼柱梁板楼面"
    },
    "混凝土品种": {
      "type": "string",
      "description": "表格右上方的「混凝土品种」字段，如：泵送混凝土"
    },
    "强度等级": {
      "type": "string",
      "description": "表格右下区域每行「强度等级」字段，统一提取为主等级，如：C30"
    },
    "施工日期列表": {
      "type": "array",
      "items": {
        "type": "string"
      },
      "description": "所有明细行中的「施工日期」去重后组成的数组，如：[\"2021\"]"
          }
  },
  "required": ["检测日期", "检测原因", "检测方法", "检测部位", "混凝土品种", "强度等级", "施工日期列表"]
}
```

### 字段说明

| 字段名 | 数据类型 | 示例值 | 含义说明 | 提取位置 |
|--------|----------|--------|----------|----------|
| 检测日期 | string | 2024-10-10 | 检测执行的日期 | 表格右上角「检测日期」 |
| 检测原因 | string | 委托检测 | 进行检测的原因 | 表格中部靠右 |
| 检测方法 | string | 回弹法 | 使用的检测方法 | 表格中部靠右 |
| 检测部位 | string | 2#楼柱梁板楼面 | 被检测的工程部位 | 「工程名称」或「检测部位」字段 |
| 混凝土品种 | string | 泵送混凝土 | 混凝土的类型 | 表格右上方 |
| 强度等级 | string | C30 | 混凝土强度等级（主值） | 表格右下区域每行「强度等级」 |
| 施工日期列表 | array | ["2021"] | 所有施工日期（去重） | 明细行中的「施工日期」字段 |

### 提取规则

1. **强度等级**：从表格右下区域的每行「强度等级」字段中提取，统一提取为主等级（如 C30）
2. **施工日期列表**：提取所有明细行中的「施工日期」字段值，去重后组成字符串数组
3. **缺失字段**：若字段缺失或无法读取，设置为 `null`

## 混凝土强度检测表格（KSQR-4.13-XC-10）

### JSON Schema

```json
{
  "type": "object",
  "properties": {
    "控制标号": {
      "type": "string",
      "description": "表格左上角控制编号，用于表格类型判断，如：KSQR-4.13-XC-10"
    },
    "设计强度等级": {
      "type": "string",
      "description": "混凝土设计强度等级，如：C30"
    },
    "检测部位": {
      "type": "string",
      "description": "混凝土构件位置描述，如：一层柱3/1/B"
    },
    "混凝土品种": {
      "type": "string",
      "description": "混凝土类型，如：泵送混凝土"
    },
    "检测日期": {
      "type": "string",
      "description": "表格右上区域，如：2024-10-14"
    },
    "施工日期": {
      "type": "string",
      "description": "表格右上区域，如：2021-01-01"
    },
    "碳化深度（mm）": {
      "type": "number",
      "description": "表格中「碳化深度」列的「计算」值，数值类型，如：6.00"
    },
    "测区强度最小值（MPa）": {
      "type": "number",
      "description": "表格底部字段，数值类型，如：33.7"
    },
    "测区强度平均值（MPa）": {
      "type": "number",
      "description": "表格底部字段，数值类型，如：35.7"
    },
    "测区强度标准差（MPa）": {
      "type": "number",
      "description": "表格底部字段，数值类型，如：1.19"
    },
    "混凝土强度推定值（MPa）": {
      "type": "number",
      "description": "最终计算结果，数值类型，如：33.7"
    }
  },
  "required": ["控制标号", "设计强度等级", "检测部位", "混凝土品种", "检测日期", "施工日期", "碳化深度（mm）", "测区强度最小值（MPa）", "测区强度平均值（MPa）", "测区强度标准差（MPa）", "混凝土强度推定值（MPa）"]
}
```

### 字段说明

| 字段名 | 数据类型 | 示例值 | 含义说明 | 提取位置 |
|--------|----------|--------|----------|----------|
| 控制标号 | string | KSQR-4.13-XC-10 | 表格左上角控制编号，用于表格类型判断 | 表格左上角 |
| 设计强度等级 | string | C30 | 混凝土设计强度等级 | 表格中 |
| 检测部位 | string | 一层柱3/1/B | 混凝土构件位置描述 | 表格中 |
| 混凝土品种 | string | 泵送混凝土 | 混凝土类型 | 表格中 |
| 检测日期 | string | 2024-10-14 | 检测执行的日期 | 表格右上区域 |
| 施工日期 | string | 2021-01-01 | 施工日期 | 表格右上区域 |
| 碳化深度（mm） | number | 6.00 | 碳化深度计算值 | 表格中「碳化深度」列的「计算」值 |
| 测区强度最小值（MPa） | number | 33.7 | 测区强度最小值 | 表格底部字段 |
| 测区强度平均值（MPa） | number | 35.7 | 测区强度平均值 | 表格底部字段 |
| 测区强度标准差（MPa） | number | 1.19 | 测区强度标准差 | 表格底部字段 |
| 混凝土强度推定值（MPa） | number | 33.7 | 最终计算的强度推定值 | 表格底部字段 |

### 提取规则

1. **控制标号**：优先识别字段，位于表格左上角，用于判断表格类型
2. **碳化深度**：提取表格中「碳化深度(mm)」列的「计算」值（不是测值），数值类型，单位为mm
3. **数值字段**：所有带"（MPa）"或"（mm）"的字段应为数字类型，不要带单位
4. **缺失字段**：若字段缺失或无法读取，设置为 `null`
