# 表格类型说明

本文档描述了混凝土表格识别技能支持的所有表格类型。

## 支持的表格类型

### 1. 混凝土回弹检测记录表（结构物检测原始记录）

- **类型代码**: concrete_strength_sheet
- **完整名称**: 混凝土回弹检测记录表（结构物检测原始记录）
- **描述**: 结构混凝土强度回弹检测，施工记录回溯、AI解析报告生成
- **用途**: 结构混凝土强度回弹检测，施工记录回溯、AI解析报告生成

#### 识别特征

- **优先识别方式**：控制标号位于表格左上角，当控制标号为 `KJQR-056-215` 时，识别为该类型
- **次级识别特征**：
  - 表头包含「构件检测原始记录」或「结构检测」「回弹法」
  - 出现字段「检测部位」「混凝土品种」「检测方法」「强度等级」「施工日期」
  - 关键词：「KJQR-056-215」「构件检测」「现场结构检测」「回弹法」「构件检测原始记录」「结构检测」

#### 字段列表

- 检测日期
- 检测原因
- 检测方法
- 检测部位
- 混凝土品种
- 强度等级
- 施工日期列表

---

### 2. 混凝土强度检测表格（KSQR-4.13-XC-10）

- **类型代码**: concrete_strength_grid
- **完整名称**: 回弹法检测混凝土强度原始记录（KSQR-4.13-XC-10）
- **描述**: 用于结构实体混凝土强度回弹检测的原始数据采集
- **用途**: 用于结构实体混凝土强度回弹检测的原始数据采集

#### 识别特征

- **优先识别方式**：控制标号位于表格左上角，当控制标号为 `KSQR-4.13-XC-10` 时，识别为该类型
- **次级识别特征**：
  - 网格结构的回弹值矩阵
  - 出现「推定值 MPa」「平均值 MPa」「标准差」等统计项
  - 检测依据字段如「JGJ/T 23-2011」
  - 关键词：「KSQR-4.13-XC-10」「推定值」「平均值」「标准差」

#### 字段列表

- 控制标号
- 设计强度等级
- 检测部位
- 混凝土品种
- 检测日期
- 施工日期
- 碳化深度（mm）
- 测区强度最小值（MPa）
- 测区强度平均值（MPa）
- 测区强度标准差（MPa）
- 混凝土强度推定值（MPa）

## 表格识别流程

1. **图片输入**: 支持PDF和图片格式（PNG, JPG等）
2. **类型分类**: 使用千问API快速识别表格类型
3. **数据提取**: 根据表格类型加载对应的Schema，提取结构化数据
4. **格式化输出**: 支持JSON、CSV、Excel格式输出

## 使用说明

- 每个表格类型的字段定义和Schema请参考 `table_schemas.md`
- 字段提取规则将在提供示例表格后详细定义
