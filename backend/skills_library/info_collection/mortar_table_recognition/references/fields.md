# Mortar Strength Inspection Fields
# 砂浆强度检测字段定义

本文档定义了砂浆强度检测报告中需要抽取的字段。

## 1. 表头字段（meta）

### 1.1 table_id (控制编号/表格ID)

- **位置**: 左上角
- **类型**: string
- **必填**: 是
- **示例**: `KJQR-056-206`
- **说明**: 表格的控制编号或唯一标识

### 1.2 record_no (原始记录编号)

- **位置**: 右上角 "No: xxxxx"
- **类型**: string
- **必填**: 是
- **示例**: `2500108`
- **说明**: 原始记录的编号

### 1.3 test_date (检测日期)

- **类型**: date/string
- **必填**: 是
- **格式**: YYYY-MM-DD
- **示例**: `2023-02-26`
- **规范化规则**:
  - 优先输出 YYYY-MM-DD 格式
  - 若无法可靠规范化（手写模糊/缺日/月），则保留原字符串
  - **禁止推断补全**

### 1.4 instrument_model (仪器型号)

- **类型**: string | null
- **必填**: 否
- **示例**: `SJY-800B`
- **说明**: 检测使用的仪器型号

---

## 2. 表格数据行（rows）

每一行代表一个检测记录，包含以下字段：

### 2.1 seq (序号)

- **列位置**: 序号列（如存在）
- **类型**: integer | null
- **必填**: 否
- **示例**: `1`
- **规则**:
  - 能识别就输出
  - 识别不到允许 null
  - **不允许自动补号/重排**

### 2.2 test_location (检测部位)

- **列位置**: **第一列**（以列语义为准，非固定序号）
- **类型**: string
- **必填**: 是
- **示例**: `一层墙 19×D-F 轴`
- **规范化规则**:
  - 去首尾空格
  - 把连续空格压成1个
  - **不改写语义**
  - **不拆分楼层/轴线字段**

### 2.3 converted_strength_mpa (砂浆抗压强度换算值)

- **列位置**: **倒数第二列**（以列语义为准）
- **类型**: number | null
- **必填**: 否
- **单位**: MPa
- **精度**: 保留1位小数
- **示例**: `2.0`
- **规则**:
  - 若为空、为"—"或"//"等占位符：输出 null
  - 若为数值：保留1位小数

### 2.4 estimated_strength_mpa (单个构件强度推定值)

- **列位置**: **倒数第一列**（以列语义为准）
- **类型**: number | null
- **必填**: 否
- **单位**: MPa
- **精度**: 保留1位小数
- **示例**: `1.8`
- **规则**:
  - 若为空、为"—"或"//"等占位符：输出 null
  - 若为数值：保留1位小数

---

## 3. 数据规范化规则

### 3.1 日期格式

- **标准格式**: YYYY-MM-DD
- **示例**: 2024-01-24
- **特殊情况**: 无法规范化时保留原字符串，禁止推断补全

### 3.2 数值格式

- **类型**: number | null
- **精度**: 保留1位小数
- **空值处理**: 空、"—"、"//"等占位符统一输出 null
- **示例**: `2.0`, `1.8`, `null`

### 3.3 整数格式

- **类型**: integer | null
- **规则**: 能识别就输出，识别不到允许 null
- **禁止**: 自动补号或重新排序

### 3.4 文本格式

- **清洗规则**:
  - 去除首尾空格
  - 压缩连续空格为1个空格
- **禁止**: 改写语义内容

---

## 4. 字段验证规则

### 4.1 必填字段检查

以下字段必须存在且不为 null:
- `meta.table_id`
- `meta.record_no`
- `meta.test_date`
- `rows[].test_location` (每行)

### 4.2 数值范围验证

- `converted_strength_mpa`: 通常在 0.5 - 20.0 MPa 之间
- `estimated_strength_mpa`: 通常在 0.5 - 20.0 MPa 之间
- 超出范围时在 notes 中说明

### 4.3 列位置识别

**重要**: 以下字段按**列语义**识别，而非固定列序号:
- `test_location` = 第一列（检测部位描述列）
- `converted_strength_mpa` = 倒数第二列（换算值列）
- `estimated_strength_mpa` = 倒数第一列（推定值列）

---

## 5. 缺失值处理

- 使用 `null` 表示缺失值
- 不要使用空字符串 `""` 或其他占位符
- 在 `notes` 字段中记录缺失原因（如适用）

---

## 6. 特殊情况说明

### 6.1 手写模糊

- 日期模糊：保留原样，不推断
- 数字模糊：尽量识别，不确定记录在 notes

### 6.2 表格变体

- 有些表格可能多列或少列
- 严格按列语义（第一列、倒数第二列等）识别
- 不要依赖固定的列序号

### 6.3 占位符识别

以下视为空值，输出 null:
- `—` (破折号)
- `//` (双斜杠)
- 空白单元格
- 其他明显的占位符号
