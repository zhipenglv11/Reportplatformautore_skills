# 软件计算结果识别

## 用途
- 识别复核验算相关参数
- 输出结构化 JSON
- 可直接入库并用于报告章节“荷载及计算参数取值”

## 提取字段
- `mortar_strength_mpa`：砌筑砂浆抗压强度取值
- `brick_strength_grade`：砌墙砖抗压强度等级（如 `MU10.0`）
- `live_loads`：活载分项
- `dead_loads`：恒载（含自重）分项
- `load_combination_type`：荷载基本组合类型（如 `1.2D+1.4L`）

## 默认值（未识别到时）
- 砌筑砂浆抗压强度取值：`1.1 MPa`
- 砌墙砖抗压强度等级：`MU10.0`
- 活载：
  - 不上人屋面：`0.5 kN/m²`
  - 客厅、卧室、厨房、卫生间：`2.0 kN/m²`
  - 楼梯、阳台：`2.5 kN/m²`
- 恒载（含自重）：
  - 屋面：`4.0 kN/m²`
  - 楼面（预制板）：`3.0 kN/m²`
  - 楼梯间：`6.0 kN/m²`
- 荷载基本组合类型：`1.2D+1.4L`
